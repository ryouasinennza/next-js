# next.js starter kit

## node manager install

```bash
curl https://get.volta.sh | bash
source ~/.zshrc

# https://volta.sh
```

## node_modules install

```bash
pnpm install
```

## dev

`pnpm dev`

## build

`pnpm build`

## start

`pnpm start`
